
# File Organizer Automation Tool

## Description
A Python script that automatically organizes files into folders based on their type (images, documents, videos).

## Features
- Automatically categorizes files
- Supports multiple file types
- Reduces manual effort
- Simple and efficient

## Technologies Used
- Python
- OS module
- Shutil module

## How to Run
1. Clone the repository
2. Run the script:
   python main.py
3. Enter the folder path when prompted

mixed files in one folder
